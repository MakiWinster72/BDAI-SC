# 班级审核

「班级审核」供班干部审核本班成员提交的信息修改申请。

> 仅 **班干部** 可访问此页面。

## 功能入口

在顶部导航栏点击「班级审核」，进入审核工作台。

## 待审核列表

左侧列出所有待审核的申请，按时间倒序排列。

每条申请显示：

- **申请人姓名**
- **申请类型**（信息修改 / 成果提交）
- **提交时间**
- **简短摘要**

## 审核详情

点击申请进入详情视图：

### 信息修改申请

展示修改的字段对比：

```
手机号码
原值：138xxxx1234
新值：139xxxx5678
```

审核操作：**通过** 或 **驳回**（需填写驳回理由）

### 成果提交申请

展示成果基本信息、描述、媒体附件。

审核操作：**通过** 或 **驳回**（需填写驳回理由）

## 批量审核

如有多条待审核申请，可在列表页勾选后点击「批量通过」或「批量驳回」。

批量驳回时需统一填写驳回理由。

## 审核历史

已完成审核的申请移至「已审核」Tab，可查看审核结果和意见，不可再修改。
