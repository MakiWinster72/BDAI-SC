# 通知中心

「通知中心」集中展示所有审核状态更新和系统消息。

![通知中心](/images/notification审核.png)

## 类型

| 类型     | 说明                       |
| -------- | -------------------------- |
| 成果审核 | 成果提交后的审核结果通知   |
| 信息修改 | 个人资料变更的审核结果通知 |

## 分类

有五个分类： 未读、待处理、已滞后、已通过、已驳回

未读中会搜集其余四个分类所有未读通知

待处理会显示： 1. 当前账号申请中的审核；2. 被分配到的审核通知（管理员全部， 教师/班干部为所分配的班级）

待处理中过久未处理的审核，超过2天（默认）后会变更到已滞后

> 管理员可在 `后台管理` > `其他设置`，里面调整变更的时间间隔

## 未读标记

收到新通知后，顶部导航栏的通知入口会显示红色未读计数。

> 此已读标记只记录在当前浏览器本地缓存，所以更换浏览器登录/更换设备，并不会同步已读
