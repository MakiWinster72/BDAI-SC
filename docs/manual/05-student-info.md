# 学生信息管理

「学生信息管理」供教师和管理员按班级查看学生档案信息。

> 仅 **教师** 和 **管理员** 可访问此页面。

## 班级选择

顶部下拉选择班级，选择后加载该班级所有学生的列表。

## 学生列表

以数据表格形式展示，列包括：

- **姓名**
- **学号**
- **手机号码**
- **成果总数**
- **最后提交时间**

支持按姓名或学号搜索。

## 查看学生详情

点击学生姓名或「查看」按钮，进入学生档案详情页。

详情页展示该学生的完整档案：

- 基本信息（姓名、学号、性别、民族、政治面貌等）
- 联系信息（手机、邮箱、地址）
- 教育经历列表
- 干部经历列表
- 个人成果列表（可按类型筛选）

## 教师权限说明

- **可查看**：所有学生档案的完整信息
- **不可编辑**：教师无修改权限，信息修改由学生自行提交并经班干部审核

## 管理员权限说明

- 与教师相同，可查看所有学生档案
- 管理员还可通过「后台管理」直接修改任意学生信息
