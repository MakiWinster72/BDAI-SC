# 个人成果

「个人成果」用于录入、查看和管理个人各类成果。

![个人成果总览页](/images/achievements个人成果总览页.png)

## 成果类型

目前有以下九类：

| 类型 | 说明                                 |
| ---- | ------------------------------------ |
| 竞赛 | 学科竞赛、创新创业比赛等             |
| 论文 | 发表的期刊论文、会议论文             |
| 期刊 | 期刊投稿与发表                       |
| 专利 | 发明专利、实用新型专利、外观设计专利 |
| 证书 | 专业技能证书、语言证书等             |
| 科研 | 学生参与教师科研项目                 |
| 作品 | 艺术作品、软件作品、设计作品等       |
| 双百 | 双百工程                             |
| 大创 | 大学生创新创业训练计划项目           |

## 新增成果

点击右下角加号按钮，选择成果类型，填写表单：

![添加成果页面](/images/achievements个人成果-添加成果.png)
相比原 xlsx 表格，可多添加图片与附件，图片可存放例如竞赛过程中的记录，团队合照，学习历程等。附件可添加类似荣誉证书，记录视频等

> 管理员可在后台管理限制图片与附件的格式、数量与大小

## 编辑与删除

点击任意成果卡片进入详情页
![查看成果详情](/images/achievements个人成果查看.png)

在详情页可「编辑」修改内容，或「删除」整条记录

## 提交审核

成果编辑完成后，点击「提交审核」将成果送交审核流程。
